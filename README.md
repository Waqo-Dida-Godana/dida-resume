## 🚀 Installation

To run locally:

```bash
# Clone the repository
git clone https://github.com/Waqo-Dida-Godana/dida-resume.git

# Navigate to the project folder
cd dida-resume

# Install dependencies
npm install

# Start the development server
npm start
